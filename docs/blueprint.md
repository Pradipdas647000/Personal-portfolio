# **App Name**: Aetherfolio

## Core Features:

- Dynamic Hero Section: Fullscreen hero area with floating glass card, displaying key information, avatar, typing animation, and providing quick access to other sections (Projects, Contact, Resume).
- Project Showcase: A premium bento-grid layout featuring floating glassmorphism cards for each project, including images, tech stack, descriptions, and links to GitHub and live demos.
- Developer Bio & Timeline: An 'About' section presented as a large floating holographic glass panel, detailing Pradip's introduction, education, development journey, and a futuristic timeline of experience.
- Interactive Skills Display: Showcase technical skills (React, Next.js, Node.js, etc.) using floating glass cards arranged in a modern grid layout with hover effects.
- Contact & Resume Module: A minimal, futuristic floating contact panel with links to GitHub, LinkedIn, email, a resume download button, and a contact form with glowing inputs.
- Responsive & Adaptive Design: Ensure the 'Minimal Floating Glass World' aesthetic and functionality are consistent and visually appealing across various devices and screen sizes.

## Style Guidelines:

- Primary interactive elements will utilize a luminous violet-blue (HSL(240, 60%, 50%), Hex: #6640FF), reminiscent of deep space glows, contrasting against dark backgrounds for visibility.
- The background will be a desaturated deep charcoal with a subtle violet hint (HSL(240, 15%, 10%), Hex: #18151D), providing a 'deep black' canvas for the floating elements.
- Accent elements will be highlighted with a bright, crisp cyan (HSL(210, 80%, 70%), Hex: #64DBFF), creating 'neon cyan highlights' that draw attention to key interactive components.
- Headline text will use 'Space Grotesk' (sans-serif) for its computerized, futuristic character, while body text will use 'Inter' (sans-serif) for clean readability and a modern, neutral feel.
- Modern, minimalistic line-based icons will be used, complemented by subtle glowing effects and floating motions to match the futuristic, glassmorphism theme, particularly in the contact section.
- The layout features floating transparent glass panels, rounded glass cards, and a bento-grid for projects, all with soft 3D depth, premium spacing, blur backgrounds, and responsive adaptation for an elegant hierarchy.
- Smooth cinematic animations are integral, including Framer Motion transitions, floating motion, mouse parallax, fade/scroll reveal, hover interactions with glow pulse, and dynamic blur transitions for an immersive and fluid user experience.
