
# 🚀 Pradip Das | Full Stack Developer Portfolio

A premium, futuristic personal portfolio website showcasing my technical skills, innovative projects, and professional journey as a Full Stack Developer.

🌐 **Live Demo:** [personal-portfolio-beta-eight-48.vercel.app](https://personal-portfolio-beta-eight-48.vercel.app)

---

## 📌 About The Project

This portfolio is built to demonstrate a high level of UI/UX design combined with modern frontend architecture. It features smooth glassmorphism effects, interactive 3D-tilt animations, and a fully responsive layout.

### ✨ Key Features
- **Dynamic Hero Section:** Featuring a typewriter effect and interactive mouse-tracking card.
- **Project Showcase:** A filtered grid of featured creations with tech stack badges and live demos.
- **Experience Timeline:** A vertical layout detailing professional and academic milestones.
- **Technical Arsenal:** A visual grid of languages, frameworks, and tools.
- **Contact Integration:** Functional contact form powered by EmailJS.
- **Mobile Optimized:** Custom navigation menu with solid, high-visibility styling.

---

## 🛠️ Tech Stack

### Frontend
- **React.js & Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Shadcn/UI** (Components)
- **Lucide React** (Icons)

### Tools & Backend
- **Firebase** (Project Foundation)
- **EmailJS** (Form Submissions)
- **Git & GitHub**

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pradipdas647000/Personal-portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd Personal-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run development server:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   `http://localhost:3000`
